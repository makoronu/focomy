# マニュアル生成

**対象読者: 境界域知能でも使える**

---

## フロー

```
1_prepare（準備）
├── pages.md       ← 対象ページ定義
└── terms.md       ← 用語変換辞書

2_analyze（分析）
├── code.md        ← コードから機能抽出
└── screenshot.md  ← スクショ（検証用のみ）

3_approve（承認）
└── review.md      ← ユーザー承認

4_generate（生成）
├── text.md        ← 説明文生成
└── simplify.md    ← 文章簡素化

5_output（出力）
└── export.md      ← HTML/PDF出力（スクショなし）
```

---

## ルール

- 1動作1ステップ
- スクショは検証用（マニュアルに含めない）
- コードから機能を網羅的に抽出
- 生成前にユーザー承認必須
- 判断させない
- 専門用語ゼロ
- 小4レベル文章
