# Focomy 公式テーマ計画

## テーマ一覧

| テーマ名 | 用途 | 特徴 |
|---------|------|------|
| Default | 汎用 | 現在のテーマ。標準的なレイアウト |
| Minimal | ポートフォリオ・個人 | 余白多め、タイポグラフィ重視 |
| Corporate | 企業サイト | ヘッダー大きめ、CTA重視 |
| Blog | ブログ特化 | サイドバー、カテゴリ、アーカイブ |

---

## 共通仕様

### 必須テンプレート

```
themes/{theme_name}/
├── templates/
│   ├── base.html           # ベーステンプレート
│   ├── home.html           # トップページ
│   ├── post.html           # 投稿詳細
│   ├── post_list.html      # 投稿一覧
│   ├── page.html           # 固定ページ
│   ├── category.html       # カテゴリページ
│   ├── archive.html        # アーカイブ
│   ├── search.html         # 検索結果
│   └── 404.html            # エラーページ
├── static/
│   ├── css/
│   │   └── style.css
│   └── js/
│       └── main.js
└── theme.yaml              # テーマ設定
```

### theme.yaml

```yaml
name: minimal
label: Minimal
description: シンプルでミニマルなテーマ
version: 1.0.0
author: Focomy Team
preview: preview.png

# カスタマイズ可能な設定
config:
  colors:
    primary:
      type: color
      default: "#2563eb"
      label: メインカラー
    background:
      type: color
      default: "#ffffff"
      label: 背景色
    text:
      type: color
      default: "#1e293b"
      label: 文字色

  typography:
    font_family:
      type: select
      options:
        - system-ui
        - "Noto Sans JP"
        - "Hiragino Sans"
      default: system-ui
      label: フォント
    base_size:
      type: number
      default: 16
      min: 14
      max: 20
      label: 基本文字サイズ

  layout:
    max_width:
      type: number
      default: 1200
      label: 最大幅(px)
    sidebar:
      type: boolean
      default: true
      label: サイドバー表示

  header:
    logo_height:
      type: number
      default: 40
      label: ロゴ高さ(px)
    sticky:
      type: boolean
      default: true
      label: 固定ヘッダー

# ウィジェットエリア
widget_areas:
  - sidebar
  - footer_1
  - footer_2
  - footer_3

# メニュー位置
menu_locations:
  - header
  - footer
```

## 1. Default テーマ（現在）

### 特徴
- 標準的な2カラムレイアウト
- ダークモード対応
- レスポンシブ

### 改善点
- theme.yaml 追加
- 設定でカスタマイズ可能に
- ウィジェット対応

---

## 2. Minimal テーマ

### コンセプト
- 余白を活かしたデザイン
- コンテンツにフォーカス
- アニメーション控えめ

### レイアウト
```
┌─────────────────────────────────────┐
│          [Logo]                      │ ← 中央配置
├─────────────────────────────────────┤
│                                      │
│    ┌─────────────────────────┐      │
│    │                          │      │
│    │      コンテンツ          │      │
│    │      (最大680px)         │      │
│    │                          │      │
│    └─────────────────────────┘      │
│                                      │
├─────────────────────────────────────┤
│              Footer                  │
└─────────────────────────────────────┘
```

### 配色
- 白背景 + 黒文字
- アクセントカラー1色のみ
- リンクはアンダーライン

## 4. Blog テーマ

### コンセプト
- 記事の読みやすさ重視
- カテゴリ・タグのナビゲーション
- 関連記事・人気記事

### レイアウト
```
┌─────────────────────────────────────┐
│ [Logo]  [カテゴリ1] [カテゴリ2] ... │
├─────────────────────────────────────┤
│                                      │
│  ┌────────────────┐  ┌──────────┐  │
│  │                 │  │ 検索     │  │
│  │  記事一覧       │  ├──────────┤  │
│  │                 │  │ カテゴリ │  │
│  │  ┌───────────┐ │  ├──────────┤  │
│  │  │ 記事カード │ │  │ 人気記事 │  │
│  │  └───────────┘ │  ├──────────┤  │
│  │  ┌───────────┐ │  │ タグ     │  │
│  │  │ 記事カード │ │  ├──────────┤  │
│  │  └───────────┘ │  │ アーカイブ│  │
│  │                 │  │          │  │
│  └────────────────┘  └──────────┘  │
│                                      │
├─────────────────────────────────────┤
│              Footer                  │
└─────────────────────────────────────┘
```

### 特徴
- カテゴリをヘッダーに表示
- サイドバーにウィジェット
- 記事カードレイアウト
- 無限スクロールまたはページネーション

## CSS設計

### 方針
- CSS変数でカスタマイズ
- BEM命名規則
- モバイルファースト

### 共通変数
```css
:root {
  /* テーマ設定から注入 */
  --color-primary: #2563eb;
  --color-background: #ffffff;
  --color-text: #1e293b;
  --font-family: system-ui;
  --font-size-base: 16px;
  --max-width: 1200px;
}
```

## 実装順序

| 順番 | テーマ | 理由 |
|-----|-------|------|
| 1 | Default | 既存テーマの整理・theme.yaml対応 |
| 2 | Minimal | シンプルで実装しやすい |
| 3 | Blog | 需要が高い |
| 4 | Corporate | 複雑だが需要あり |

## テーマ切り替え

### Admin UI
`/admin/appearance/themes`
- インストール済みテーマ一覧
- プレビュー
- 有効化
- カスタマイズ

### config.yaml
```yaml
theme:
  active: minimal
  settings:
    primary: "#2563eb"
    sidebar: true
```
