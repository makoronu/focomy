# Focomy シンプル化方針

## 原則

**必要最小限の機能で最大の価値を**

---

## 削除対象

### サイドバー
| 機能 | 理由 |
|------|------|
| ウィジェット | モダンサイトでは使わない |
| コメント | Disqus等外部サービスで代替 |
| サイトマップ | 自動生成で十分、管理画面不要 |
| リンク検証 | 優先度低い |

### 編集画面
| 機能 | 理由 |
|------|------|
| 抜粋 | 本文から自動生成 |
| OGタイトル | SEOタイトルと統合 |
| OG説明文 | SEO説明文と統合 |
| OG画像 | アイキャッチと統合 |

---

## 残す機能

### サイドバー
- ダッシュボード
- コンテンツ（固定ページ、タグ、シリーズ、チャンネル）
- メディア
- 外観（テーマ、メニュー）
- 設定（ユーザー、サイト設定、リダイレクト）
- ツール（インポート、システム情報）

### 編集画面
- タイトル
- スラッグ
- 本文
- アイキャッチ
- ステータス
- 公開日時
- SEOタイトル（→OGタイトルも兼ねる）
- SEO説明文（→OG説明文も兼ねる）
- noindex / nofollow / canonical
- 著者 / チャンネル / シリーズ / タグ

---

## 実装方針

1. OG系フィールドを削除し、SEO/アイキャッチを流用
2. 抜粋は本文先頭から自動生成（保存しない）
3. 不要なcontent_types削除（widget, comment）
4. サイドバーから不要リンク削除
